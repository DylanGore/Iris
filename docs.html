Raspberry Pi Initial Setup

  1. This guide assumes that you have already setup network and users
  2. Update the Pi
    $ sudo apt update -y && sudo apt upgrade -y
  3. Configure the Pi
    1. $ sudo raspi-config
    2. Enable the camera module
      go to Interfacing Options and enable the camera module.
    3. Expand the file system
      go to Advanced options and Expand file system.
    4. Reboot
      $ sudo reboot now
  4. Update the firmware
    $ sudo rpi-update
  5. Reboot
    $ sudo reboot now
